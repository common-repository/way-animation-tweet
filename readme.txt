WAY Animation Tweets plugin for WordPress allows you to easily show tweets in real time from three most popular twitter resources. Tweets can be shown using the included shortcode generator in posts and pages, using the widget or directly in theme using php function. The template used is highly customisable you can change the tweet layout, link colors or create your own custom template.

Features--------------
Multiple Tweet Instances
Multiple Browser Support
Responsive design
Beautiful Animation Effects
Fully customizable
Show tweets via Shortcodes and Widget
Number of tweets to show
Three popular twitter resources
List Tweets

Search Tweets
